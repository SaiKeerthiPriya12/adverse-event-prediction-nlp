
 # Adverse Event Prediction using NLP

## 📌 Description
This project predicts adverse events in healthcare datasets using Natural Language Processing (NLP).
It handles imbalanced data using data augmentation techniques to improve accuracy.

## 📌 Technologies Used
- Python
- NLP (TF-IDF Vectorizer)
- Machine Learning
- Flask

## 📌 Project Files
- app.py – Main application
- adverse_event_model.pkl – Trained model
- tfidf_vectorizer.pkl – Vectorizer
- requirements.txt – Dependencies
- trained_model.ipynb – Model training

## 📌 How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   python app.py

## 📌 Output
The system predicts whether input text indicates an adverse event or not.
