# adverse-event-prediction-nlp
Enhancing adverse event prediction using NLP-based data augmentation in imbalanced healthcare datasets.
Mini Project - B.Tech AIML
 Description:
This project focuses on predicting adverse events in healthcare datasets using Natural Language Processing (NLP) techniques. It handles imbalanced data by applying data augmentation methods to improve prediction accuracy.
 Technologies Used:
Python
NLP (TF-IDF Vectorizer)
Machine Learning
Flask (for web app)
 Files in Project:
app.py – Main application file
adverse_event_model.pkl – Trained ML model
tfidf_vectorizer.pkl – Text vectorizer
requirements.txt – Required libraries
trained_model.ipynb – Model training notebook
 How to Run:
Install requirements
pip install -r requirements.txt
Run the app
python app.py
 Output:
The system predicts whether a given input text indicates an adverse event or not.
